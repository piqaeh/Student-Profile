UiTM Student E - Profile

University Technology MARA

The E-Student Profile (E-SP) platform is a web-based system for managing student profiles, designed with a user-friendly interface. It features quick navigation to sections like Home, Report, and Contact, along with a dark mode toggle for a personalized view. Users can filter by department, faculty, or project and search for specific profiles. Representing Universiti Teknologi MARA (UiTM), it displays the institution's logo and provides profile statistics, showing 11,650 completed and 50 incomplete profiles. A highlight section showcases featured students with their names and photos, making it a practical tool for organizing and presenting student information.
